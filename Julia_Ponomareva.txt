1. Открываем Git Bash
2. $ ls (просматриваем дирректорию где мы находимся)
3. $ cd "название папки" (обязательно должны находиться в ветке main)
4. $ git status (проверяем есть ли что нам commit и где мы находимся. Видим что есть файл который изменился: modified: "название файла")
5. $ git add . (файл добавляется в индекс)
6. $ git status (проверяем статус нашего файла: изменение цвета с красного на зеленый)
7. $ git commit -m "комментарий" (фиксируем изменения)
8. $ git push (отправляем на сервер)
9. $ git log (проверяем все записанные commit, в том числе и наш)
